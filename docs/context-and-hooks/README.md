# Контекст и хуки — глубокая справка

Как контекст проходит через число «пи», как его перехватывать и формировать, а также расширенные шаблоны для авторов расширений.

Эти документы заполняют пробелы между документами по расширению Pi высокого уровня и фактической реализацией исходного кода. Сначала прочтите документацию по расширению Pi, чтобы узнать основы, а затем используйте ее для точной работы.
