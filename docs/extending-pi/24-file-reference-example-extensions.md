## Краткая справка: «Я хочу...»

| Цель | Подход | Ключ API | Пример файла |
|------|----------|---------|-------------|
| Блокировать опасные команды | Прослушайте `tool_call`, верните `{ block: true }` | `pi.on("tool_call", ...)` | `protected-paths.ts` |
| Добавить инструмент, который может использовать LLM | Зарегистрируйте инструмент со схемой и выполните | `pi.registerTool({...})` | `todo.ts` |
| Добавить косую черту | Зарегистрируйте команду с обработчиком | `pi.registerCommand(...)` | `commands.ts` |
| Задать вопрос пользователю | Использовать методы диалога | `ctx.ui.select()`, `ctx.ui.confirm()` | `question.ts` |
| Показать постоянный статус | Установить статус нижнего колонтитула | `ctx.ui.setStatus(id, text)` | `status-line.ts` |
| Измените системную подсказку | Крючок `before_agent_start` | Возврат `{ systemPrompt: "..." }` | `pirate.ts` |
| Фильтровать сообщения, отправленные на номер LLM | Крюк `context` событие | Возврат `{ messages: [...] }` | — |
| Сохранение состояния при перезапуске | Храните в сведениях об инструменте или appendEntry | `details: {...}` / `pi.appendEntry(...)` | `todo.ts` |
| Пользовательское уплотнение | Крючок `session_before_compact` | Возврат `{ compaction: {...} }` | `custom-compaction.ts` |
| Создайте полноэкранный режим UI | Используйте `ctx.ui.custom()` | Компонент с рендером/handleInput | `snake.ts` |
| Показать плавающее диалоговое окно | Использовать режим наложения | `ctx.ui.custom(..., { overlay: true })` | `overlay-test.ts` |
| Заменить редактор ввода | Продлить `CustomEditor` | `ctx.ui.setEditorComponent(...)` | `modal-editor.ts` |
| Переопределить встроенный инструмент | Зарегистрировать инструмент с таким же именем | `pi.registerTool({ name: "read" })` | `tool-override.ts` |
| Запускайте инструменты через SSH | Использовать подключаемые операции | `createBashTool(cwd, { operations })` | `ssh.ts` |
| Программное переключение моделей | Найти и установить модель | `pi.setModel(model)` | `preset.ts` |
| Зарегистрируйте собственного поставщика | Предоставить конфигурацию с моделями | `pi.registerProvider(...)` | `custom-provider-anthropic/` |
| Преобразование пользовательского ввода | Крюк `input` событие | Возврат `{ action: "transform", text }` | `input-transform.ts` |
| Внедрить сообщения | Отправка персонализированных или пользовательских сообщений | `pi.sendMessage()` / `pi.sendUserMessage()` | `send-user-message.ts` |
| Реагировать на изменения модели | Крючок `model_select` | `pi.on("model_select", ...)` | `model-status.ts` |
| Добавить сочетание клавиш | Зарегистрировать ярлык | `pi.registerShortcut("ctrl+x", ...)` | `plan-mode/` |
| Пакет для распространения | Добавьте ключ `pi` к package.json | `"pi": { "extensions": [...] }` | См. packages.md |
